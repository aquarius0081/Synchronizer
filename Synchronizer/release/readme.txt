Перед запуском нужно:
1) Установить Java SE 8;
2) Прописать путь до папки с установленной Java 8 в переменную среды JAVA_HOME;
3) Разархивировать все файлы данного архива в отдельную папку;
4) Установить MS SQL 2008 R2 Developer Edition;
5) Создать инстанс SQL сервера локально с именем "EMPLOYEESLIST" на стандартном порту;
6) Разрешить подключаться к этому инстансу удаленно;
7) Создать пользователя БД с именем "esa" и паролем "qweASD12#";
8) Убедиться что у данного пользователя достаточно прав чтобы развернуть бэкап Enterprise.bak;
9) Развернуть бэкап Enterprise.bak базы на ранее созданном инстансе;
10) Дать ранее созданному пользователю полные права на базу Enterprise;
11) Запустить test.bat с параметром export / sync и путем к XML файлу для экспорта / синхронизации (пример: test.bat sync import.xml);
12) Все.
